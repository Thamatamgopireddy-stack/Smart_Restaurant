🍽 Smart Food Ordering System
A QR-based restaurant ordering and tracking system.

🚀 Features
Scan QR to view menu
Place orders digitally
Track order status (Pending → Preparing → Ready → Served)
Kitchen dashboard for order management
🧠 Problem Solved
Reduces manual errors, waiting time, and improves restaurant efficiency.

🛠 Tech Stack
Frontend: React / Next.js
Styling: CSS
Deployment: CodeSandbox
